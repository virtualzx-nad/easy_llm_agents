The WeatherAPI.com offers a free weather API and geolocation developer API for commercial and non-commercial use. The API provides real-time, forecasted, future, marine, and historical weather data in JSON and XML formats. The real-time or current JSON weather and XML weather API provides access to near real-time weather information for millions of locations worldwide by global collaborations of weather stations and high-resolution local weather models. The historical and future weather API provides 14-day daily, hourly, and 15 min interval weather forecast, future weather (up to 365 days ahead), marine weather, and hourly historical weather (from 2010 onwards) for millions of cities and towns around the world in JSON and XML format. The Geolocation API provides air quality data, time zone, astronomy, IP lookup, sports API, currency API, map API, and geo API.
